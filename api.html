API

set(icon)

Set or transform icon of Marka element.
Arguments:

  • icon (string): Type of icon you want to set to Marka element.

color(colorValue)

Change icon color of Marka element.
Arguments:

  • colorValue (string): HEX or RGB formatted string to color Marka element.

size(sizeValue)

Change icon size of Marka element.
Arguments:

  • sizeValue (integer): Size of icon (in pixels).

rotate(direction)

Change icon direction of Marka element.
Arguments:

  • direction (string): Direction of icon. It can be "up", "down", "left", or "right".
